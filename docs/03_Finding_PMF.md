# 03 · 寻找 PMF：产品市场契合度的真实指标

> **目标：** 知道「有没有 PMF」之前，先定义 **对你这个阶段什么叫信号**。

---

## 一、别被融资叙事带偏

PMF 不是：

- 拿到天使轮
- 上了 Product Hunt 首页
- 朋友说你很厉害

PMF 是：**一群陌生人，持续地、自愿地，为你的价值付出代价**（时间、金钱、推荐）。

---

## 二、早期可观测信号

按阶段从弱到强排列：

| 信号 | 说明 | 适用阶段 |
|------|------|----------|
| **回访** | 7 日内主动回来第二次 | MVP |
| **推荐** | 未 incentivize 的情况下介绍朋友 | MVP+ |
| **付费** | 真钱，不是「愿意付」 | 验证定价 |
| **留存曲线趋平** | 下滑后不再归零 | 增长期 |
| **自然增长** | 不投流也有新增 | 增长期 |

**一周 MVP 阶段：** 只盯 **回访 + 推荐**，别盯 DAU 绝对值。

---

## 三、Sean Ellis 测试（简化版）

问已用过的用户：

> 「如果明天不能再使用这个产品，你会有多失望？」

- **≥40%** 选「非常失望」→ 值得深挖 PMF 信号
- **<20%** → 大概率还没打到痛点，或人群不对

样本 **<30** 时仅供参考，别过度解读。

---

## 四、Kill / Keep 决策表

每周五填一次：

| 指标 | 本周 | 上周 | 判断 |
|------|------|------|------|
| 陌生人完成核心动作数 | | | |
| 7 日回访率 | | | |
| 付费 / 预购数 | | | |
| 用户主动推荐数 | | | |

**规则：**

- 连续 3 周核心指标 **零增长** → 考虑 pivot 或换人群
- 有增长但留存差 → 修产品，不扩渠道
- 留存好但规模小 → 可以开始想分发
